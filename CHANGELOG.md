# Version 0.1.0 — Backend Foundation

**Release Date:** 10 July 2026

## Overview

This release establishes the complete backend foundation for PrepPilot. The project now includes a working Express server, REST APIs, middleware, environment configuration, professional folder structure, and a mini backend project demonstrating production-style architecture.

---

## Added

### Backend Setup

* Initialized Node.js backend.
* Configured npm and package management.
* Created Express.js server.
* Configured environment variables using dotenv.
* Added professional backend folder structure.

### HTTP & APIs

* Implemented HTTP server.
* Implemented Request and Response handling.
* Created RESTful APIs.
* Built Notes Mini Backend Project.
* Added API testing using Postman.

### Routing

* Implemented static routes.
* Implemented dynamic routes.
* Implemented route parameters.
* Introduced Express Router.

### Middleware

* Configured `express.json()`.
* Built custom middleware.
* Implemented request logging.
* Implemented route-specific middleware.

### Architecture

* Added `src` directory.
* Added `routes` folder.
* Added `controllers` folder.
* Added `middleware` folder.
* Added `config` folder.
* Added `services` folder.
* Added `utils` folder.

### Documentation

Updated:

* README.md
* Progress Tracker
* Developer Journal
* Interview Notes
* API Documentation
