# 🚀 PrepPilot

PrepPilot is a full-stack AI-powered interview preparation platform built using modern web technologies.

The platform aims to help students and aspiring software engineers prepare for technical interviews through:

- 🎯 AI-powered mock interviews
- 📄 Resume management and analysis
- 📊 Progress tracking and performance insights
- 📝 Personalized study notes and learning plans
- 💡 Interview question practice and preparation tools
